# Talking Chloe ❤️

Talking Chloe is an AI-powered conversational assistant built using Django, designed to deliver personalized, emotionally engaging chat experiences with a 3D interactive interface.

## 🚀 Features

- 🤖 AI Chat (Groq - LLaMA 3)
- 🌐 Multilingual Support (Hindi, Indian English, US English)
- 🔐 User Authentication (Login & Register)
- 💬 Chat History with Memory
- 🎭 Personality-based AI (Flirty Chloe Persona)
- 📷 Camera Context Awareness (Mood & Outfit Color)
- 🎨 3D Model Interaction (GLB Support)

---

## 🛠️ Tech Stack

- Python (Django)
- HTML, CSS, JavaScript
- Groq API (LLM)
- SQLite Database
